## Inputs vs Outputs

### Inputs
Inputs are the data provided to a machine learning model, usually represented as features.

### Outputs
Outputs are the predictions or decisions produced by the model based on the inputs.

**Goal**  
Training aims to learn an accurate mapping from inputs to outputs.
