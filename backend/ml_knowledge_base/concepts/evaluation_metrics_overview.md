## Evaluation Metrics

Evaluation metrics quantify how well a trained model performs on data.

Different tasks use different metrics:
- Regression focuses on prediction error magnitude
- Classification focuses on decision correctness

**Purpose**  
Metrics provide an objective way to compare models.
